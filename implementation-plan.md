# Counter PWA 実装計画

## 1. プロジェクトのセットアップ

### 1.1 Expo 環境構築

- Expo 環境のセットアップ（expo-cli 使用）
- TypeScript 設定
- 必要なパッケージのインストール
  - expo-web-browser（PWA サポート）
  - expo-haptics（振動フィードバック）
  - expo-av（音声フィードバック）
  - @expo/webpack-config（PWA 設定用）

### 1.2 プロジェクト構成

```
counter-expo/
├── src/
│   ├── components/
│   │   ├── ConfirmDialog.tsx
│   │   └── SettingsPanel.tsx
│   ├── constants/
│   │   └── index.ts
│   ├── hooks/
│   │   ├── useFeedback.ts
│   │   └── useStorage.ts
│   └── App.tsx
├── assets/
└── app.json
```
